Screen Orientation API

Open console. 

// screen.onmozorientationchange - событие смены ориентации экрана устройства
// screen.mozOrientation - текущее значение ориентации экрана
screen.addEventListener(
    "mozorientationchange",
    function() {
        console.log("the orientation of the device is now " + screen.mozOrientation);
    }
);
